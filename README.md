# Bingo

[Polymer](https://www.polymer-project.org/1.0/) game experiment by [Alejandro Arroyo Duque](http://alexarroyoduque.github.io/).

## Development

### Install dependencies

```sh
$ bower install && npm install
```

### Tasks

```sh
$ gulp
```

```sh
$ gulp build
```
