# Bingo

Demo: [https://master-bingo.web.app](https://master-bingo.web.app)


## Quick start

- node version: 14.18.3

```
npm install -g polymer-cli
git clone "this-repository"
cd "this-project"
npm install
polymer serve
```

## Deploy

```
polymer build
polymer serve build/default
```

```
firebase deploy
```
